Sunnah Tracker App

This is a clean, semantic HTML5 landing page designed for a mobile application.

Key Features: Includes a navigation bar with internal anchor links, an app interface gallery using placeholder imagery, a video demo section with an embedded YouTube iframe, and a functional "Join Waitlist" registration form.

Tech Used: Pure HTML5 for structure and internal page navigation.

Academy Cinemas Site

A responsive, modern home page for a movie theater built using the Bootstrap 4 framework.

Key Features: Utilizes a fixed navigation bar, a fluid Jumbotron for the hero section, and a dynamic "card-deck" layout to display movie showtimes with badge-pill styling.

Tech Used: HTML5, Bootstrap 4 (CSS/JS), and jQuery.
